The gitignore file

Each line in a gitignore file specifies a pattern.

You can place a gitignore file in the parent directory or the same directory as the path.

Here are some common patterns:

*.css.map - Ignore all files with .css.map file extension composer.lock - Ignore the composer.lock /vendor - Ignore the vendor directory and all files in it
